# Blog Preview Card

## Overview
This project is a responsive blog preview card that displays an avatar, title, and short description of a blog post. The primary focus of this project was on **CSS styling** and creating a clean, readable card layout that works well across different screen sizes.

This was one of my earlier **Frontend Mentor** challenges and played an important role in strengthening my understanding of layout and visual hierarchy.

## Why This Project Matters
This project helped reinforce foundational front-end skills, particularly around structuring content and improving design consistency. It also encouraged more intentional thinking about how small UI components contribute to overall user experience.

## Features
- Responsive card layout
- Clean typography and spacing
- Mobile-friendly design
- Clear visual hierarchy

## What I Learned
- Using **Flexbox** and positioning to structure UI components
- Applying **media queries** to improve mobile responsiveness
- Managing and reusing **multiple CSS classes** effectively
- Improving visual hierarchy through spacing and alignment

## Challenges
One of the main challenges was implementing media queries in a way that improved the mobile layout without breaking the structure or readability of the card on larger screens.
